# ZSH_Custom_Functions

Some customized functions for zsh in archlinux and bspwm : ) at your risk

Adding functions to separate files in ZShell (Zsh) offers several important advantages:

Organization and Modularity
Separate functionalities into logical modules

Each file can group related functions

Facilitates maintenance and navigation

Conditional Loading (Autoloading)

Reusability

Functions can be shared between scripts

Easy import into different configurations.

----------------------------------------------

- pyenv -> Python check, activate and create virtual environment, can add dependencies check, or modify with venv used.

- rubenv -> Ruby check, check, activate and create virtual environment, can add dependencies check, or modify with venv used.

- portkill -> Kill used port and check perms to kills. Verify port proccess.

- tools -> Speed search with help of fzf, in customized folder.

- list-functions -> List functions in functions file, (sintax function name(){....}
